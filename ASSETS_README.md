# BEEP FILMS - Asset Setup

## Required Assets

### 1. Hero Background Video
- **Location**: `public/videos/hero-video.mp4`
- **Description**: Full-screen background video for the homepage
- **Recommended specs**:
  - Format: MP4 (H.264)
  - Resolution: 1920x1080 or higher
  - Duration: 10-30 seconds (will loop)
  - File size: Optimized for web (< 10MB recommended)

### 2. Animated Icon/GIF
- **Location**: `public/images/animated-icon.gif`
- **Description**: Small circular animated icon that appears next to "GET UNFILTERED" text
- **Recommended specs**:
  - Format: GIF or APNG
  - Size: 80x80 pixels (circular)
  - Background: Transparent or white

## Adding Your Assets

1. Place your hero video in: `/public/videos/hero-video.mp4`
2. Place your animated icon in: `/public/images/animated-icon.gif`
